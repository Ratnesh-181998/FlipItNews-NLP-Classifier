## üé® Screenshot Guidelines

### Quality
- Use high resolution (1920x1080 or higher)
- Ensure text is readable
- Capture full tab content
- Use light mode for better visibility

### Naming Convention
- Use lowercase with underscores
- Be descriptive
- Use .png format

### Tools
- **Windows**: Snipping Tool, Snip & Sketch, or Win+Shift+S
- **Mac**: Command+Shift+4
- **Browser Extension**: Awesome Screenshot, Nimbus
